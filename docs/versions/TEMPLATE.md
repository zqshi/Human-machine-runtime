# Snapshot 模板（版本完成后）

版本完成时，删除"任务详情"和"实施要点"等执行细节，只保留:

1. 版本目标
2. 交付内容摘要
3. 关键技术决策表
4. 遗留问题
5. 对后续版本的约束
6. 质量检测结果摘要（见 CLAUDE.md §14）
