## 1. 功能概述

### 1.1 功能定位

租户管理是运营管理后台的核心功能模块，位于左侧导航栏「租户管理」分组下。平台管理员通过该模块管理所有租户（企业/组织）的生命周期：从创建租户并分配初始管理员，到日常编辑、暂停/激活，直至删除。本期聚焦租户基础信息管理与状态流转，配额/资源/AI 用量等高级配置作为后续迭代。

### 1.2 目标用户

| 用户类型 | 特征描述 | 核心诉求 |
|---------|---------|---------|
| 平台管理员 | 运维人员，拥有平台管理权限 | 全量管理租户，包括创建、状态控制、删除 |

### 1.3 核心价值

| 维度 | 描述 |
|------|------|
| 用户价值 | 一站式管理所有租户，无需登录数据库手动操作 |
| 业务价值 | 租户开通从小时级降至分钟级，初始管理员自动生成密码降低沟通成本 |

### 2.3 详细规格

#### F-01：租户列表

**功能描述**: 以表格形式展示所有租户，每行包含租户名称（含 slug）、当前状态、操作按钮。

**前置条件**:
- 用户已以平台管理员身份登录
- 当前处于「租户管理」导航项

**页面结构**:
- 页面标题区：左侧"租户管理"标题 + 副标题"租户生命周期管理与配额分配"，右侧"新建租户"按钮 + 刷新按钮
- 表格区：圆角卡片内嵌表格

**表格列**:

| 列名 | 内容 | 说明 |
|------|------|------|
| 租户 | 名称（粗体）+ slug（小灰字） | 名称和 slug 上下排列 |
| 状态 | 状态标签 | 带颜色的圆角标签 |
| 操作 | 操作按钮组 | 根据状态动态展示 |

**操作按钮按状态展示规则**:

| 当前状态 | 可操作 |
|---------|--------|
| active | 编辑（图标）、暂停 |
| suspended | 编辑（图标）、激活、删除 |
| archived | 编辑（图标）、删除 |

**状态标签颜色**:

| 状态 | 颜色 |
|------|------|
| active | 绿色背景+绿色文字 |
| suspended | 黄色背景+黄色文字 |
| archived | 灰色背景+灰色文字 |

**业务规则**:
- 表格无数据时，居中显示"暂无租户"
- 点击租户行打开详情 Drawer
- 点击操作按钮不触发行点击事件

**边界条件**:

| 边界场景 | 处理方式 |
|---------|---------|
| 无租户 | 居中显示"暂无租户" |
| 加载中 | 居中显示"加载中..." |
| 租户名称过长 | 自然截断 |

#### F-02：新建租户

**功能描述**: 点击"新建租户"按钮，右侧滑出 Drawer，填写租户基础信息和初始管理员信息，提交后创建租户并弹出凭证 Drawer。

**前置条件**:
- 用户已以平台管理员身份登录

**主要行为**:
1. 点击页面右上角"新建租户"按钮
2. 右侧滑出 Drawer（宽 560px），标题"新建租户"
3. Drawer 顶部有 Tab 栏，本期仅启用"基础信息"Tab
4. 基础信息包含三个区块：租户信息 + 公司信息 + 管理员

**表单字段 — 租户信息区块**:

| 字段 | 必填 | 说明 |
|------|------|------|
| 名称 | 是 | 租户名称 |

> 注：本期不提供套餐选择（已注释预留）

**表单字段 — 公司信息区块**:

| 字段 | 必填 | 类型 | 说明 |
|------|------|------|------|
| 行业 | 否 | 下拉选择 | 选项：金融科技、电商零售、医疗健康、教育培训、制造工业、科技互联网、其他 |
| 公司规模 | 否 | 下拉选择 | 选项：初创（<20人）、小型（20-99人）、中型（100-499人）、大型（500-999人）、超大型（1000+人） |
| 联系电话 | 否 | 文本输入 | placeholder："公司联系电话" |
| 公司描述 | 否 | 多行文本 | 3行高度，不可拖拽调整，placeholder："简要描述公司业务..." |

**表单字段 — 管理员区块**:

| 字段 | 必填 | 说明 |
|------|------|------|
| 管理员 | 是 | 管理员姓名 |
| 邮箱 | 是 | 管理员邮箱 |
| 初始管理员账号 | — | 独立区块，详见下方 |

**初始管理员账号区块**:
- 默认勾选"自动生成密码"复选框
- 勾选时：展示提示文字"系统将自动生成用户名和密码，创建成功后将显示初始登录凭证"
- 取消勾选时：展示用户名输入框 + 密码输入框（密码至少 6 个字符）

**提交行为**:
1. 点击 Drawer 底部"保存"按钮
2. 校验必填项，不通过则在按钮上方展示红色错误提示
3. 提交中按钮文案变为"保存中..."并禁用
4. 创建成功后关闭编辑 Drawer，打开凭证 Drawer 展示初始账号密码
5. 创建失败则在按钮上方展示"保存失败，请稍后重试"

#### F-03：编辑租户

**功能描述**: 点击租户行的编辑图标，右侧滑出 Drawer，加载当前租户信息并允许修改。

**前置条件**:
- 用户已以平台管理员身份登录

**主要行为**:
1. 点击租户行的编辑图标（铅笔图标）
2. 右侧滑出 Drawer（宽 560px），标题"编辑租户"
3. 表单预填当前租户信息
4. 修改可编辑字段后点击"保存"
5. 保存成功后关闭 Drawer，列表刷新

**业务规则**:
- 编辑模式与新建模式共用同一 Drawer 组件，标题区分"新建租户"/"编辑租户"
- 编辑时不展示初始管理员账号区块

---

#### F-04：暂停/激活

**功能描述**: 对租户执行暂停或激活操作，改变租户状态。

**前置条件**:
- 用户已以平台管理员身份登录
- 租户当前状态允许目标操作

**主要行为**:
1. 点击操作按钮（暂停/激活）
2. 系统执行状态变更
3. 列表自动刷新，更新租户状态和操作按钮

**业务规则**:
- 暂停（active → suspended）：操作按钮为红色背景"暂停"
- 激活（suspended → active）：操作按钮为绿色背景"激活"
- 状态变更后操作成功时，列表刷新并在右下角展示绿色成功提示（3 秒后消失）
- 状态变更失败时，在右下角展示红色错误提示

---

#### F-05：删除租户

**功能描述**: 删除租户记录，需先校验可删除性并通过二次确认。

**前置条件**:
- 租户状态为 suspended 或 archived

**主要行为**:
1. 点击租户行的"删除"按钮（红色背景）
2. 系统校验租户是否可删除
3. 根据校验结果展示不同内容

**校验不可删除时**:
- 右侧滑出 Drawer（宽 400px），标题"确认删除"
- 展示红色错误图标 + "无法删除该租户"
- 展示不可删除原因（如"租户下仍有 N 名成员"）
- 仅有"确定"按钮可点击关闭

**校验可删除时**:
- 右侧滑出 Drawer（宽 400px），标题"确认删除"
- 展示黄色警告图标 + "确定要删除此租户吗？"
- 展示租户名称
- 展示小字"此操作不可恢复，请谨慎操作。"
- 底部两个按钮："取消"（灰色）+ "确认删除"（红色）
- 确认删除后，关闭 Drawer，列表刷新，右下角展示绿色"删除成功"提示

#### F-06：租户详情

**功能描述**: 点击租户行，右侧滑出详情 Drawer。

**前置条件**:
- 用户已以平台管理员身份登录

**主要行为**:
1. 点击表格中租户行
2. 右侧滑出 Drawer（宽 520px），标题为租户名称

**详情内容区块**:

| 区块 | 展示字段 | 说明 |
|------|---------|------|
| 基本信息 | ID、Slug、状态 | 键值对列表，ID 使用等宽字体 |
| 公司信息 | 行业、公司规模、联系电话、描述 | 编辑时录入的公司信息 |
| 管理员 | 管理员、邮箱 | — |

---

#### F-07：初始凭证展示

**功能描述**: 新建租户成功后，右侧滑出凭证 Drawer，展示自动生成的初始管理员用户名和密码。

**前置条件**:
- 租户创建成功
- 系统自动生成了初始管理员账号

**主要行为**:
1. 编辑 Drawer 关闭后，凭证 Drawer 自动滑出（宽 400px）
2. 标题"租户创建成功"
3. 绿色成功提示："租户创建成功，以下是初始管理员登录凭证"
4. 用户名：只读输入框 + 复制图标按钮
5. 初始密码：只读输入框（明文）+ 复制图标按钮
6. 黄色警告："请妥善保管此密码！首次登录后请立即修改密码。"
7. 底部"确定"按钮，点击后关闭 Drawer 并刷新列表

**业务规则**:
- 点击复制图标按钮将对应内容写入剪贴板
- 凭证 Drawer 可通过右上角关闭按钮或底部"确定"按钮关闭
- 关闭后无法再次查看该密码

### 4.4 异常分支

| 异常场景 | 触发条件 | 处理方式 | 用户提示 |
|---------|---------|---------|---------|
| 网络异常 | 请求超时或服务不可达 | 右下角 Toast | 红色错误提示 |
| 保存失败 | 创建/编辑提交失败 | Drawer 不关闭，按钮上方红色错误 | "保存失败，请稍后重试" |
| 不可删除 | 租户有成员 | 删除 Drawer 展示不可删除原因 | 红色"无法删除该租户"+ 原因 |
| 校验状态失败 | 检查可删除性时出错 | 右下角 Toast | "检查租户状态失败" |
| 空状态 | 无租户数据 | 表格居中文字 | "暂无租户" |

## 5. 租户状态流转

### 5.1 状态定义

| 状态 | 含义 | 标签颜色 |
|------|------|---------|
| active | 正常运行 | 绿色 |
| suspended | 已暂停 | 黄色 |
| archived | 已归档 | 灰色 |

### 5.2 状态流转图

```
[active] ──[暂停]──→ [suspended] ──[激活]──→ [active]
                        │
                        │──[删除]──→ ∅（删除成功）
                        
[archived] ──[删除]──→ ∅（删除成功）
```

### 5.3 操作与状态对应表

| 当前状态 | 可执行操作 |
|---------|-----------|
| active | 编辑、暂停 |
| suspended | 编辑、激活、删除 |
| archived | 编辑、删除 |
