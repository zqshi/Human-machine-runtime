## 1. 功能概述

### 1.1 功能定位

登录页面是运营管理后台的唯一入口，负责身份验证与会话建立。本期定位为 MVP：采用单一固定管理员账号 `admin/admin123`，不涉及多角色权限分配、SSO 集成、用户注册等逻辑。后续迭代将接入 RBAC 与 SSO。

### 1.2 目标用户

| 用户类型 | 特征描述 | 核心诉求 |
|---------|---------|---------|
| 运营管理员 | 平台运维人员，负责租户管理、系统配置等 | 快速登录后台，执行管理操作 |
| 开发调试人员 | 本地开发/测试环境使用者 | 简单账密即可进入系统 |

### 1.3 核心价值

| 维度 | 描述 |
|------|------|
| 用户价值 | 零配置登录，输入固定账密即可进入管理后台 |
| 业务价值 | MVP 阶段快速交付，为租户管理等后续功能提供认证基座 |

### 2.3 详细规格

#### F-01：账密登录

**功能描述**: 用户在登录页输入用户名和密码，点击"登录"按钮完成认证，成功后进入工作台页面。

**前置条件**:
- 系统已预置管理员账号（admin/admin123）
- 用户未处于已登录状态

**主要行为**:
1. 用户访问管理后台，未登录时展示登录页
2. 登录页居中展示一个登录卡片，包含品牌信息和登录表单
3. 用户输入用户名、密码，点击"登录"
4. 系统验证账密通过，进入工作台页面
5. 验证失败则在密码框下方展示错误文字

**业务规则**:
- 本期仅 admin 账号可登录，不提供注册/找回密码入口
- 用户名和密码均为必填，任一为空时"登录"按钮禁用
- 登录中按钮文案变为"登录中..."并不可重复点击
- 密码以密文形式输入，不支持切换明文
- 支持 Enter 键提交表单

**登录卡片品牌信息**（运营管理平台）:

| 元素 | 内容 |
|------|------|
| 图标 | domain 图标，靛蓝渐变背景 |
| 标题 | 运营管理平台 |
| 副标题 | 租户运营与平台监控 |
| 标签 | 多租户管理、平台配置、审计日志 |

**登录后进入的工作台**:
- 三栏布局：Dock（80px 图标导航栏）| Sidebar（可调宽度面板）| Main（主内容区）
- Dock 底部显示用户头像（取 displayName 首字母）
- 点击头像弹出菜单，展示用户显示名和用户ID

**边界条件**:

| 边界场景 | 处理方式 |
|---------|---------|
| 用户名或密码为空 | "登录"按钮禁用 |
| 用户名或密码错误 | 密码框下方显示红色错误文字 |
| 系统服务不可达 | 密码框下方显示错误信息 |
| 连续登录失败 | 每次均展示错误提示，本期不做限流/锁定 |

---

#### F-02：登出

**功能描述**: 用户点击 Dock 底部头像，在弹出菜单中点击"退出登录"。

**前置条件**:
- 用户已登录

**主要行为**:
1. 用户点击 Dock 底部的头像图标（显示 displayName 首字母）
2. 弹出菜单，展示：用户显示名、用户ID、"个人资料"、"设置"、"退出登录"
3. 点击"退出登录"（红色文字），系统清除会话，回到登录页

**业务规则**:
- 点击菜单外部区域自动关闭菜单
- 登出为即时生效，不二次确认

---

#### F-03：登录失败提示

**功能描述**: 登录请求失败时，在登录表单的密码框下方展示一行红色错误文字。

**主要行为**:
1. 账密错误 → 展示具体错误信息（如"用户名或密码错误"）
2. 用户再次点击登录时，先清除上一次的错误信息

**业务规则**:
- 错误信息不透露具体是用户名错误还是密码错误（防止枚举攻击）
- 错误信息不会自动消失，仅在用户重新提交时清除

---

#### F-04：会话恢复

**功能描述**: 用户打开页面时，系统自动尝试恢复已有的登录态。

**主要行为**:
1. 用户打开页面，系统检测到已有有效会话
2. 页面居中展示"正在恢复会话…"文字
3. 恢复成功后直接进入工作台，无需重新登录
4. 恢复失败则展示登录页

## 4. 交互流程

### 4.1 登录主流程（时序图）

```
用户                 登录页               系统
 │                    │                   │
 │── 访问管理后台 ───→│                   │
 │                    │── 检测登录状态 ──→│
 │                    │←── 未登录 ────────│
 │                    │                   │
 │                    │── 展示登录卡片 ──→│（含品牌信息+表单）
 │                    │                   │
 │── 输入用户名 ─────→│                   │
 │── 输入密码 ───────→│                   │
 │── 点击"登录" ────→│                   │
 │                    │── 提交认证 ──────→│
 │                    │                   │
 │                    │    ┌── 认证结果 ──┤
 │                    │    │              │
 │                    │←── 成功 ──────────│
 │←── 进入工作台 ─────│                   │
 │                    │                   │
 │                    │    或             │
 │                    │                   │
 │                    │←── 失败 ──────────│
 │←── 显示错误文字 ───│                   │
```

### 4.2 登出流程（时序图）

```
用户                 工作台               系统
 │                    │                   │
 │── 点击Dock底部头像→│                   │
 │                    │── 弹出菜单 ──────→│
 │── 点击"退出登录" →│                   │
 │                    │── 清除会话 ──────→│
 │←── 回到登录页 ─────│                   │
```

### 4.3 会话恢复流程（时序图）

```
用户                 页面                 系统
 │                    │                   │
 │── 打开页面 ───────→│                   │
 │                    │── 尝试恢复会话 ──→│
 │                    │                   │
 │                    │    ┌── 会话状态 ──┤
 │                    │    │              │
 │←── "正在恢复会话…" ─│                  │
 │                    │                   │
 │                    │←── 有效 ──────────│
 │←── 进入工作台 ─────│                   │
 │                    │                   │
 │                    │←── 无效 ──────────│
 │←── 展示登录页 ─────│                   │
```

### 4.4 异常分支

| 异常场景 | 触发条件 | 处理方式 | 用户提示 |
|---------|---------|---------|---------|
